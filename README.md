# 🎓 Student Record Management System 2 1 3 1

A Python console-based application for managing student records with JSON persistence.

## 🚀 Features
- Add, update, delete, and view students
- Sort by name or marks
- Persistent data using JSON
- CSV export/import
- Logging and exception handling
- Optional Streamlit GUI

## 🧠 Tech Stack
- Python (Core, JSON, CSV)
- Streamlit (Optional)
- Logging & Unit Testing (pytest)
